# icy_term

# Features 

 * Telnet only so far 
 * Ansi, Avatar, Petscii display
 * Built in fonts, many DOS fonts, C64, Amiga & Atari fonts supported.
 * Scrollback buffer
 * Protocols supported Xmodem, Xmodem1k, Xmodem1k-G, Ymodem, Ymodem-G, Zmodem, ZedZap (Zmodem 8k)
 * IEMSI auto login support
